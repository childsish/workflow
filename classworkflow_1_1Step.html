Detailed Description

A single step in a workflow containing inputs and outputs. Steps can be piped directly to each other if they have only one output and/or input.

Member Function Documentation

Pipe the single output of this step to the single input of another. If there are multiple outputs or inputs, then an exception will be thrown.
